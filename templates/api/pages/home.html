Currency Exchange API

Welcome to the Currency Exchange API

This application allows you to:

  • Register and login to get access to the API
  • Check your balance (each user starts with 1000 coins)
  • Get currency exchange rates to UAH
  • View your exchange rate request history

Each exchange rate request costs 1 coin from your balance.

Getting Started

To begin using the API:

  1. Go to the Register/Login page to create an account or login
  2. Once logged in, check your Balance
  3. Make currency exchange requests on the Currency Exchange page
  4. View your request history on the History page
